# Local Terminus: a Rain World region mod

---

### *Venture beyond the Chasing Wind superstructure...*

This mod is currently in development.
To run the development version of it (this repository), please install [Extra Parameters](https://steamcommunity.com/sharedfiles/filedetails/?id=3557598109), as it is a required dependency. You must also install [Chasing Wind](https://steamcommunity.com/sharedfiles/filedetails/?id=3232063592) and all its dependencies. Then you may clone the repository into your `StreamingAssets/mods` directory.
